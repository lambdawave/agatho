# AGATHO AI Demo

A concept dashboard that showcases how AGATHO AI can help dropshipping brands manage influencer outreach with AI-assisted conversations.

## Getting started

Open `index.html` in your browser to explore the interactive prototype. No build tools are required.

## Features

- Gradient-forward layout inspired by the existing landing page aesthetic.
- Influencer pipeline with reach, engagement, and status indicators.
- Real-time style metrics for active campaigns, total messages, and response rate.
- AI messaging workspace with preset prompts, regeneration, and send logging.
- Context-aware copy suggestions tailored to each influencer profile.
